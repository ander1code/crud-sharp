# CRUD-Sharp--sqlite

1 - Description:
  - Prototype of CRUD in C# with SQLite. Uses the concept of database transaction.
  
2 - This CRUD has:
  - Login (SHA512)
  - Hide Icon (Toolbar)
  - Registration's Form
  - Search for Name and Code
  - Reports 
	
3 - Requirements for project's edition:
  - Visual Studio 2013
  - .net Framework 4.5.1 
  - SQLite (Binarys)
  - ADO.NET (ADO.NET provider for SQLite).
